## 甚麼是備份系統
**是不是會害怕伺服器被搞?**

**這時候備份系統就有用囉!!**

**他可以幫助你還原伺服器之前的所有設置!!!**

## 如何設置備份
**首先到打指令的地方輸入**

`/備份設置`
