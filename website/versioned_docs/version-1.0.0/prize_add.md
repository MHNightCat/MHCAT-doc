## 扭蛋獎池增加

**如果你想要對扭蛋系統增加獎品**

**可以使用**

`/扭蛋獎池增加 獎品名稱: 機率: 獎品代碼:`

**這邊說一下各個的選項**

- **機率: 這是使用百比的喔例如:`1`就是`1%` `0.1`就是`0.1%`**
- **獎品代碼: 例如Discord Nitro連結 steam連結等**

**這邊以:**

`/扭蛋獎池增加 獎品名稱:test 機率:50 獎品代碼:testestes`

**進行講解**

**打上之後就會看到:**

![](https://cdn.discordapp.com/attachments/991337796960784424/997442102571044884/unknown.png)

**這樣就代表成功囉**
