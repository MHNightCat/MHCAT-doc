## 甚麼是統計系統?
**統計系統就是將伺服器資訊以語音頻道或是文字頻道呈現**

**像這樣子**

![](https://media.discordapp.net/attachments/991337796960784424/991337807329116310/unknown.png)

## 那要如何使用呢?

第一次設置(第一次跟後面的設置不太一樣)只需要再打指令的地方打上

`/統計系統創建 統計頻道類型:`

範例:

`/統計系統創建 統計頻道類型:語音頻道`

![](https://media.discordapp.net/attachments/986970967383347240/991338484977639554/unknown.png)

這樣就完成囉

這邊特別說明一下，所有的頻道名稱都是可以自行更改的(不要動數字的部分就好了)

至於類別的話是完全可以自定義的
