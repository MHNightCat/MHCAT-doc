## 公告發送dsadsasa
**在進行公告頻道設置之前，你必須先有一個文字頻道。這邊以`#公告`當範例。<br/>
輸入以下指令:**<br/>
`/公告頻道設置 頻道:[你的頻道]`<br/>
而我這邊就是輸入<br/>
`/公告頻道設置 頻道:#公告`<br/>
**輸入完成後即可打上按enter**<br/>
**正常情況會出現這個消息**<br/>
![](https://media.discordapp.net/attachments/986161213727723520/988055725936894072/2022-06-19_202025.png)<br/>
>如果出現`你輸入的並不是一個文字頻道`，代表你輸入的不是一個公告頻道或是語音頻道<br/>

## 更改公告頻道 
**如果想要更改的話一樣**<br/>
**輸入以下指令:**<br/>
`/公告頻道設置 頻道:[你的頻道]`<br/>
**輸入完成後就成功進行更改囉!**
